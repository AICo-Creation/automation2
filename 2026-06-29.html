2026.06.29 / AI Co-Creation Briefing

クアルコムがAIソフト企業Modularを約39億ドルで買収。AIが“どのチップでも動く”時代へ——NVIDIAの「CUDAの壁」に風穴

「AIといえばNVIDIA」——ここ数年、私たちはなんとなくそう思ってきました。実際、世界中の生成AIのほとんどが、NVIDIAのチップと、その上で動く「CUDA（クーダ）」という独自ソフトの上で走っています。でも今、その“当たり前”に小さな、しかし大きな風穴が開きはじめました。スマホ向け半導体で知られるクアルコムが、AIソフトの新興企業「Modular（モジュラー）」を約39億ドル（およそ6,000億円）で買収。Modularが持つのは、「一度書けば、どのメーカーのチップでも動く」AIの土台です。今日のニュースの本質は、また一つ大型買収があった、という話ではありません。AIが、特定メーカーのチップに縛られた“借りもの”から、どこでも動かせる“選べる道具”へ——その潮目を、やさしくほどいていきます。

何が起きたのか

2026年6月、半導体大手クアルコムが、AIソフトの新興企業Modularを約39億ドル（およそ6,000億円）の全額株式で買収すると発表しました。Modularは、伝説的なソフト技術者クリス・ラトナーらが立ち上げた会社で、社員はおよそ150人。手がけているのは派手なAIモデルではなく、その“足回り”——プログラミング言語「Mojo（モジョ）」と、AIを動かす推論エンジン「MAX」です。

この足回りが、なぜそんなに高く評価されたのか。理由は一つ。MAXを使うと、同じAIのコードを、NVIDIAだけでなくAMD・インテル・クアルコムなど、別々のメーカーのチップの上で、書き直さずに動かせるからです。これまでAIは、NVIDIAのCUDAという独自ソフトに合わせて作り込むのが事実上の前提でした。Modularの技術は、その「NVIDIA以外では動かしづらい」という見えない壁（ベンダーロックイン）を、ソフトの力で溶かしにいく挑戦です。

そして見逃せないのが、この買収の狙いです。クアルコムは同じ時期に、データセンター向けAIチップ「Dragonfly」の計画や、大手との供給契約も明らかにしました。つまり今回の買収は、「自社のチップでも最先端AIをすんなり動かせる」入り口を手に入れる一手。AIの主戦場が、チップの性能比べだけでなく、「どのチップでも動かせるソフトをどちらが握るか」へ広がりはじめたことを、はっきり映し出しています。

なぜ「どのチップでも動く」が難しいのか——CUDAという見えない壁

「コードなんて、どのチップでも動いて当然では？」と思うかもしれません。でも、実はここに長年の壁があります。AIは最終的に、チップが分かる“機械語”に翻訳されて動きますが、その言葉づかい（命令セット）がメーカーごとに違うのです。NVIDIA向けに丁寧に書いたコードは、AMDやインテルのチップではそのまま動かず、書き直しが要る。だからAIの作り手は、いちばん普及したNVIDIAのCUDAに合わせて作る——こうして「NVIDIAに集まるほど、ますます離れにくくなる」循環ができあがりました。これが、性能だけでは崩せないNVIDIAの“堀（ほり）”の正体です。

Modularの発想は、ここに“共通の翻訳層”を一枚かませること。下の図のように、これまでは「NVIDIA向けに書く→NVIDIAでしか動かない」一本道でした。Mojo／MAXは、いったんメーカーに依存しない形で書いてから、それぞれのチップ用へ自動で訳し分ける。だから作り手は一度書くだけで、動かす先を後から選べる。会社にたとえれば、特定の取引先専用の伝票しか作れなかったのを、どの取引先にも出せる共通フォーマットに変えるようなものです。縛りをほどき、選択肢を取り戻す——それが今回の技術の核心です。

なぜ重要なのか

このニュースが大切なのは、AIの世界の力関係が「いちばん速いチップを誰が作るか」から「どのチップでもAIを動かせる“通り道”を誰が握るか」へと動きはじめたことを、はっきり示したからです。AIが特定メーカーのチップに縛られている間は、私たちは料金も供給もそのメーカーの都合に左右されます。世界的なAIチップ不足のなかで、「欲しくても手に入らない」「値段が言い値」という不自由を、多くの企業が味わってきました。でも“どこでも動く”土台が広がれば、手に入るチップで、いちばん割安な選択肢を選べる。これは単なる技術の話ではなく、AIという道具を、特定の一社に握られすぎないという安心の話なのです。

しかもこの流れは、もう例外ではありません。2026年に入り、クアルコムだけでなく各社が、独自ソフトの壁を越える「共通の翻訳層」づくりに本腰を入れはじめました。チップの選択肢が増えれば、競争が生まれ、AIを動かすコストと電力はさらに下がっていきます。最先端AIが、電気や水道のように“どの蛇口からでも引ける”土台（インフラ）へ近づいている——今日のニュースは、その静かで大きな地殻変動を、私たちに知らせています。

人間側に問われること

「AIがどのチップでも動くなら、もう何も気にしなくていい」とまでは言えません。なぜなら、選択肢が増えるほど、“選ぶ責任”も私たちの側に返ってくるからです。一社にすべてお任せしていれば、迷う必要はありませんでした。けれど「どの基盤に乗るか」を選べる時代は、裏を返せば「乗り換えやすさ（身軽さ）を、最初から設計しておく」賢さが問われる時代でもあります。便利さに飛びつく前に、「この仕組みは、後から別のチップやサービスへ移せるか」を一度立ち止まって確かめる——その視点が、開かれた時代ほど効いてきます。

もう一つ大切なのは、“流行りの一社”だけに頼りきらない目です。強い企業の製品はたしかに便利ですが、それ一本に組織やくらしを預けきると、価格や方針の変更にまるごと振り回されます。とはいえ、過度に身構える必要はありません。大事なのは、「最強かどうか」だけでなく「いざというとき乗り換えられるか」も合わせて選ぶこと。開かれた選択肢が増えるほど、最後に頼りになるのは、私たち自身の選ぶ力なのです。

私たちにできること

今日からできる小さな一歩は、AIを「どこか一社のサービス」ではなく「選んで・組み替えられる道具」として見はじめることです。難しい技術を学ぶ必要はありません。たとえば仕事で使うAIツールを一つ選ぶときに、「これしか使えなくなったら困らないか」「似た機能の別の選択肢はあるか」を、頭の片隅で考えてみる。それだけで、特定の流行に流されず、「自分（や組織）にとって本当に合うAIはどれか」を主体的に選ぶ側に立てます。

AIが“どのチップでも動く”時代は、私たちが技術の囲い込みに振り回される時代ではありません。むしろ「開かれた選択肢のなかから、自分の目的に合うものを、自分で選び・組み替えられる」、成熟したつき合い方の始まりです。覚えておきたいのはただ一つ——AIはもう、一社に縛られた“借りもの”ではなく、あなたが選べる道具になりはじめている。その変化を歓迎しながら、好奇心と落ち着きの両方を手に、今日もAIと一緒に歩んでいきましょう。

今回のニュースの核心——「なぜAIはメーカーごとのチップに縛られるのか」「その壁は、本当に越えられるのか」——を、正面から論じた論文をピックアップしました（arXiv・2026年3月公開。今日と同じ日付の論文が本稿執筆時点で確認できなかったため、ニュースのテーマにもっとも近い最新の研究を取り上げています）。Modularが挑む“どこでも動く”の土台が、研究の地図の上でくっきり見えてきます。

arXiv:2603.28793 / 2026年3月 GPU / 命令セット / 移植性 / ベンダーロックイン / ハードウェア非依存

Toward a Universal GPU Instruction Set Architecture: A Cross-Vendor Analysis of Hardware-Invariant Computational Primitives in Parallel Processors

「どのチップにも共通する土台」を探す——主要4メーカー・16種類のGPUを横断比較し、メーカーが違っても変わらない“計算のいろは”を洗い出した研究（2026年3月）

この論文は、まさに今日のニュースの「CUDAに書いたコードは、AMDやインテルのGPUではそのまま動かない」という壁が、なぜ生まれ、どうすれば越えられるのかを、根っこから見せてくれる一本です。研究は、主要4メーカー・16種類のGPUを初めて横断的に調べ、メーカーが違っても物理的に共通する“計算の基本要素（10個の不変な部品）”——足並みをそろえて動く処理や、階層的なメモリの使い方など——を特定しました。つまり、見た目はバラバラでも深いところには共通の地面があることを示したのです。

何を調べた？

NVIDIA・AMD・インテル・アップルなど主要メーカーのGPU16種類を横断比較し、設計が違ってもなお共通している計算の基本要素を体系的に洗い出した。

何がわかった？

メーカーごとに言葉づかい（命令セット）は違っても、物理の制約から生まれる“共通の10個の部品”が存在する。共通の土台の上なら移植は原理的に可能だと示した。

なぜ大事？

「一度書けば、どのチップでも動く」というModularの挑戦が、思いつきではなく理にかなっていることを裏づける。AIをメーカーの囲い込みから解く理論的な足場になる。

「ベンダーロックインは、もう変えられない宿命だ」——そんな思い込みを、この研究はやさしく崩します。表面のバラつき（メーカーごとの違い）と、深いところの共通点（物理が決める計算の基本）を切り分ける。そう発想を変えたからこそ、“どこでも動く”AIが、夢物語ではなく現実の設計目標になりました。今日のニュースと合わせて読むと、クアルコムのModular買収が一企業の思惑にとどまらず、研究が描いてきた「共通の土台を取り戻す」流れの上にあることが、くっきり見えてきます。AIの進歩は、より速いチップの競争だけでなく、同じAIを、より多くのチップへ、より自由に届ける工夫からも生まれていくのです。
